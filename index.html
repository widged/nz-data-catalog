Contributing

The data for this catalogue are sources from various lists of links. See github.com/widged/nz-data-lists for details about the listings included and the scraping process.

Many geospatial datasets are not included. They have been saved as a separate a raw listing. You can use data.govt.nz to search for them.

One of the data source is an open community feed NZ Datasets Community Feed (public google spreadsheet).

Anybody is welcome to add more links. Edit access is on request over the next month, when the catalog is still under development. It will be opened within a month or two (once the code is stable).
